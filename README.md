# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO.

## Conceitos Abordados

### 1. Redes Virtuais (Virtual Networks)
As redes virtuais são a base da infraestrutura em nuvem no Azure. Aprendemos sobre:
- Criação e configuração de VNets
- Sub-redes (subnets) e segmentação de rede
- Conectividade e roteamento entre redes
- Grupos de segurança de rede (NSG - Network Security Groups)

### 2. Balanceamento de Carga (Load Balancing)
O balanceamento de carga é essencial para distribuir o tráfego entre múltiplas instâncias. Cobrimos:
- Azure Load Balancer (Layer 4)
- Application Gateway (Layer 7)
- Distribuição de tráfego entre VMs
- Health checks e monitoramento

### 3. Recursos e Dimensionamento de Máquinas Virtuais
Configuração e otimização de recursos:
- Tipos e tamanhos de VMs
- Seleção adequada de SKUs
- Autoscaling e elasticidade
- Monitoramento de desempenho e custos

## Competências Adquiridas

- Compreensão profunda da arquitetura de redes em nuvem
- Configuração de infraestrutura de alta disponibilidade
- Otimização de recursos e custos
- Boas práticas em segurança de rede

## Conclusão

Este lab proporcionou uma visão prática de como implementar soluções de rede e computação na nuvem Azure, preparando-me para desafios mais complexos em arquitetura em nuvem.
